Описание темы проекта
ListLy — клиент-серверное приложение для ведения персональной медиатеки. 
Пользователь может находить медиаконтент, сохранять его в свою коллекцию, 
назначать статусы, добавлять оценки, заметки и группировать элементы по 
папкам.
В рамках данной работы разрабатывается серверная часть приложения. 
Она предоставляет Android-клиенту REST API, отвечает за авторизацию 
пользователей, 
хранение данных, работу с каталогом медиаконтента и поиск.
Основные возможности backend:
• регистрация и авторизация пользователей
• предоставление данных глобального каталога медиаконтента
• поиск фильмов, сериалов, аниме и игр
• ведение пользовательской медиатеки: статусы, избранное, оценки и заметки
• создание папок для группировки сохраненного контента

API.
Задачи:
1. Определить требования к серверной части приложения.

3. Выбрать и обосновать технологический стек.
4. Реализовать авторизацию пользователей.
5. Реализовать каталог медиаконтента и поиск.
6. Реализовать пользовательскую медиатеку, папки, статусы, оценки 
и заметки.
7. Организовать хранение данных и интеграцию с поисковым 
сервисом.
8. Протестировать основные сценарии работы backend.
